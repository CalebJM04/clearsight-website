# ClearSight — Product Website

Landing page for ClearSight endpoint activity intelligence platform.

## Setup for GitHub Pages

1. Create a new GitHub repo called `clearsight-website` (or `yourusername.github.io`)
2. Push all files to the `main` branch
3. Go to repo Settings → Pages → Source: Deploy from branch → main / (root)
4. Site will be live at `https://yourusername.github.io/clearsight-website/`

## Adding Screenshots

Replace the placeholder images with real screenshots from your dashboard:

- `screenshots/overview.png` — The main overview dashboard showing multiple machines
- `screenshots/machine.png` — A machine detail page with heatmap and charts
- `screenshots/flags.png` — The flag rules configuration page
- `screenshots/reports.png` — The reports generation page

Screenshots should be 16:10 aspect ratio, at least 1280×800px.

**Quick way to take them:**
1. Open your dashboard in Chrome
2. Press F12 → Toggle device toolbar
3. Set to 1280×800
4. Take a screenshot (Ctrl+Shift+P → "Capture screenshot")

## Customization

- **Email**: Replace all `hello@clearsight.io` with your real email
- **Domain**: Replace `clearsight.io` with your domain if you have one
- **Testimonials**: Replace the placeholder quotes with real customer feedback
- **Pricing**: Already matches your current pricing structure
